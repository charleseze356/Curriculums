# Curriculums

My self-built learning roadmaps — what to master for a topic, and in what order, organized on learning-science principles. Each curriculum is grounded in a course syllabus (for completeness) and a primary text (for content). Built with the `curriculum-builder` skill.

This is an orientation tool, not an encoding tool: reading a curriculum is not learning it. Encoding happens through primary resources, Socratic study, and spaced repetition.

Site layout adapted from [SuperMo0/ai-engineering-curriculum](https://github.com/SuperMo0/ai-engineering-curriculum).
