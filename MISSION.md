# Mission: AI Engineering Curriculum

## Why

To go from Python developer to hireable AI Engineer. When this curriculum is complete, the learner can apply for AI Engineer roles, pass technical interviews, and ship production AI systems on day one.

## Success looks like

- Can build, evaluate, and deploy end-to-end ML and LLM-powered systems
- Has a GitHub portfolio with 6+ production-shaped projects (one per chapter)
- Can answer AI Engineering interview questions confidently (architecture, tradeoffs, evaluation)
- Has shipped a RAG + agents + deployment capstone that is publicly accessible
- Understands the full stack: data → model → inference → monitoring → cost

## Constraints

- Assumes Python fluency — no Python fundamentals taught
- Self-paced, asynchronous — no cohort, no live instructor
- Static site delivery — all interactivity is client-side only

## Out of scope

- Python syntax and fundamentals
- Web development (HTML/CSS/JS is delivery vehicle only, not curriculum content)
- Research-track ML (no writing papers, no novel architecture design)
- Mobile or edge deployment
